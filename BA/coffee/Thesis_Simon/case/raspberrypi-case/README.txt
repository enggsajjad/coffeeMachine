# Summary

**Raspberry Pi 3 case based on 0110-M-P but does not require screws.**

Models:
* Raspberry Pi 3 Model B: Works perfectly.
* Raspberry Pi 3: Works perfectly.
* Raspberry Pi 2: Should fit. Any feedback?
* Raspberry Pi B+: Community reported that it works.

This case is also compatible with the camera module. See <a href="https://www.thingiverse.com/make:529701">hint from the community</a>

# Print Settings

Printer Brand: Ultimaker
Printer: Ultimaker 2
Rafts: No
Supports: No
Resolution: 0.1mm = Normal
Infill: 20% = Light

Notes: 
3D printed with Ultimaker 2+ and 0.4mm nozzle, Cura 2.1.2 using "Normal Quality" and Infill "Light" using 2.85mm PLA filament.

0.6mm nozzle did NOT work.
